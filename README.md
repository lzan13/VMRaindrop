VMRaindrop
========

一个 Ghost 自用主题


### 配置
然后进入`Ghost`后台 > `Code injection`在`Site Header`中插入以下代码：
```
<script>
    var enableRandomCover = true; //是否开启随机封面图，默认为true
    var enableInfiniteScroll = true; //是否开启滚动加载更多，默认为true
</script>
```
